2.自己紹介ページにCSSを適用

条件としては、文字は左寄せ、リストタグの4辺内側に10pxの幅とします。

1.模範解答

body {
    background: #f2f2f2;
}

div {
    background: #ffffff;
    width: 400px;
    padding: 20px;
    text-align: left;
    border: 1px solid #cccccc;
    margin: 30px auto;
}

li {
    padding: 10px;
}

.profiletable th,td {
    border: solid 1px #000000;
  }
